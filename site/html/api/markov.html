markov Module#

ssid.markov.okid(input, output, **options)#

Identify Markov parameters, or discrete impulse response data, for a given set of input and output data, using the Observer Kalman Identification Algorithm (OKID) (Juang, Phan, Horta, Longman, 1993).

Parameters:

  • input (array) – input time history data. dimensions: \((q,nt)\), where \(q\) = number of input channels, and \(nt\) = number of timesteps

  • output (array) – output response history data. dimensions: \((p,nt)\), where \(p\) = number of output channels, and \(nt\) = number of timesteps

  • m (int) – number of Markov parameters to compute. default: minimum of 300 and the number of timesteps in input and output.

Returns:

the Markov parameters, with dimensions \((p,q,m+1)\)

Return type:

array
